# Aprendendo Matemática com Python

## Sobre o Projeto

O projeto de Extensão **Aprendendo Matemática com Python** tem como objetivo ensinar conceitos matemáticos fundamentais e intermediários com o apoio da linguagem de programação **Python**, promovendo simultaneamente o desenvolvimento do pensamento matemático e computacional.

O material foi desenvolvido com foco educacional, priorizando clareza conceitual, exemplos práticos e exercícios progressivos. Ele é voltado principalmente para estudantes e professores que desejam utilizar Python como ferramenta de apoio ao ensino e à aprendizagem de Matemática.

---

## Objetivos

- Introduzir programação em Python de forma gradual e aplicada;
- Utilizar Python como ferramenta para exploração de conceitos matemáticos;
- Desenvolver raciocínio lógico, abstração e resolução de problemas;
- Preparar o aluno para aplicações matemáticas computacionais mais avançadas.

---

## Estrutura do Material

O conteúdo está organizado em arquivos Markdown independentes, cada um correspondente a um módulo temático.

### 1 - Fundamentos de Python

Introdução à linguagem Python, abordando:
- Variáveis e tipos de dados
- Entrada e saída
- Operadores aritméticos, lógicos e de comparação
- Estruturas condicionais (`if / elif / else`)
- Laços de repetição (`for`, `while`)
- Mensagens de erro e depuração
- Exercícios, desafios e projeto integrador

---

### 2 - Estruturas de Dados

Apresenta as principais estruturas de dados da linguagem:
- Listas
- Tuplas
- Conjuntos (sets)
- Dicionários
- Indexação e fatiamento
- Compreensões (listas, dicionários e conjuntos)

Inclui aplicações práticas, exercícios e desafios com foco em organização e manipulação de dados.

---

### 3 - Funções e Organização de Programas

Explora a criação e o uso de funções em Python:
- Definição de funções
- Parâmetros e valores de retorno
- Argumentos padrão e nomeados
- Uso de `*args` e `**kwargs`
- Escopo de variáveis
- Funções lambda
- Docstrings e *type hints* (PEP 484)

O módulo enfatiza boas práticas, reutilização de código e clareza estrutural.

---

### 4 - Tratamento de Erros e Programação Defensiva

Conteúdo dedicado à robustez de programas:
- Tratamento de exceções (`try / except / else / finally`)
- Exceções customizadas
- Uso de `assert`
- Programação defensiva
- Boas práticas para diagnóstico e falhas previsíveis

Essencial para o desenvolvimento de programas confiáveis e reutilizáveis.

---

### 5 - Arquivos e Manipulação de Dados

Aborda operações de entrada e saída de dados:
- Leitura e escrita de arquivos texto
- Arquivos CSV e JSON
- Uso do módulo `pathlib`
- Logging básico

Inclui exemplos práticos voltados a análise de dados, automação e processamento em lote.

---

### 6 - Computação Numérica com NumPy

Introdução ao uso da biblioteca NumPy para matemática computacional:
- Criação e propriedades de arrays
- Indexação e fatiamento
- Operações matemáticas vetorizadas
- Broadcasting
- Funções universais (ufuncs)
- Álgebra linear
- Estatística
- Entrada e saída de dados

Este módulo conecta diretamente programação e Matemática, sendo fundamental para aplicações científicas.
